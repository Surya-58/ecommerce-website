# 🛒 Full Stack E-commerce Website

A complete full stack e-commerce web application with user interface, backend API, and admin dashboard.

---

## 🚀 Features

- User Registration & Login
- Product Listing
- Add to Cart
- Order Management
- Admin Dashboard (Add/Edit/Delete Products)
- Responsive Design (Mobile Friendly)

---

## 🛠 Tech Stack

### Frontend
- React.js
- HTML, CSS, JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

---

## 📂 Project Structure

fullstack-ecommerce/
│── frontend/ → User website  
│── backend/ → API & server  
│── admin/ → Admin panel
